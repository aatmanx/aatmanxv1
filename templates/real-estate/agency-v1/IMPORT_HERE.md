# Paste Lovable Code Here

This folder is where exported Lovable code should be pasted.

## Workflow
1. Build your website in Lovable for this template category (agency).
2. Export the code from Lovable.
3. Paste the entire export into this `IMPORT_HERE/` folder.
4. Ask Cursor: "Convert the Lovable code in IMPORT_HERE into an Aatmanx reusable template."
5. Cursor performs conversion **inside this same template folder** — moving artifacts into `components/`, `pages/`, and `assets/`.
6. The converted template remains at `templates/real-estate/agency-v1/`.

Do not paste Lovable code into `components/` or `pages/` directly — always use `IMPORT_HERE/` first.
